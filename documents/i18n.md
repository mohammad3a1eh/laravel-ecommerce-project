# How to Use Translations (i18n) Inside TSX Components
`laravel-12` + `inertia` + `react` + `laravel-react-i18n`

## 1- Import the i18n Hook
```typescript jsx
import { useLaravelReactI18n } from 'laravel-react-i18n';
```

This gives you access to the translator function (`t`) and the current locale.

## 2- Initialize the Translator in Your Component
```typescript jsx
const { t, currentLocale } = useLaravelReactI18n();
const locale = currentLocale();
```

`t()` translates keys.<br>
`locale` tells you the active language (e.g. fa, en).

## 3- Use `t()` to Translate Any Text
```typescript jsx
<h1>{t('dashboard')}</h1>
<p>{t('welcome')}</p>
<input placeholder={t('search')} />
<Button label={t('save_changes')} />
```

Always wrap displayed text with `t('key')`.

## 4- JSON Translation Files Structure (NO PHP)
**Example:** `\lang\en.json`
```json
{
    "dashboard": "Dashboard",
    "welcome": "Welcome",
    "search": "Search...",
    "save_changes": "Save changes",
    "welcome_user": "Hello :name!"
}
```

**Example:** `\lang\fa.json`
```json
{
    "dashboard": "داشبورد",
    "welcome": "خوش آمدید",
    "search": "جستجو...",
    "save_changes": "ذخیره تغییرات",
    "welcome_user": "سلام :name!"
}
```
